Exercise 1

Consider two simple games \(G^1 = (A, v^1)\) and \(G^2 = (A, v^2)\) with the same set of players \(A\). Suppose that a player \(i \in A\) is not a null player in both games. Can we conclude that \(i\) is not a null player in the game \(G^\cap = (A, v^\cap)\), with the characteristic function \(v^\cap\) given by \begin{equation*} v^\cap(C) = \min\{v^1(C), v^2(C)\}? \end{equation*}

What about the game \(G^\cup = (A, v^\cup)\), where \(v^\cup\) is given by \begin{equation*} v^\cup = \max\{v^1(C), v^2(C)\}? \end{equation*}

Exercise 1: Solution

Neither conclusion can be drawn. We show this by producing two games, \(G^1, G^2\), in which there exists a player who is not a null player, but is a null player in \(G^\cap / G^\cup\).

Exercise 1: Solution

For the first statement, we have the following counterexample:

\(\emptyset\) \(\{1\}\) \(\{2\}\) \(\{3\}\) \(\{1,2\}\) \(\{1,3\}\) \(\{2,3\}\) \(\{1,2,3\}\)
\(v^1\) 0 0 0 0 1 0 1 1
\(v^2\) 0 0 0 0 0 1 1 1
\(v^\cap\) 0 0 0 0 0 0 1 1

Here, player \(1\) is not a null player in \(G^1\) or \(G^2\), but is a null player in \(G^\cap\).

Exercise 1: Solution

For the second statement, we have the following counterexample:

\(\emptyset\) \(\{1\}\) \(\{2\}\) \(\{3\}\) \(\{1,2\}\) \(\{1,3\}\) \(\{2,3\}\) \(\{1,2,3\}\)
\(v^1\) 0 0 0 1 1 1 1 1
\(v^2\) 0 0 1 0 1 1 1 1
\(v^\cup\) 0 0 1 1 1 1 1 1

Here, player \(1\) is not a null player in \(G^1\) or \(G^2\), but is a null player in \(G^\cup\).

Exercise 2

Prove that any outcome in the core maximizes the social welfare, i.e., for any coalitional game \(G\), it holds that if \((\mathcal{CS}, \boldsymbol{x})\) is in the core of \(G = (N, v)\), then for any coalitional structure \(\mathcal{CS}^\prime\) for \(G\), we have, \begin{equation*} \sum_{C \in \mathcal{CS}} v(C) \geq \sum_{C^\prime \in \mathcal{CS}^\prime} v(C^\prime). \end{equation*}

Exercise 2: Solution

We have, \begin{align*} \sum_{C^\prime \in \mathcal{CS}^\prime} v(C^\prime) &\leq \sum_{C^\prime \in \mathcal{CS}^\prime} \sum_{i \in C^\prime} x_i\quad&&(x\text{ satisfies the core-constraints}) \\ &= \sum_{i \in N} x_i \\ &= \sum_{C \in \mathcal{CS}} \sum_{i \in C} x_i\quad&&(\text{repartitioning}) \\ &= \sum_{C \in \mathcal{CS}} v(C)\quad &&((\mathcal{CS}, x)\text{ is an outcome}) \end{align*}

Exercise 3

Suppose an outcome \((\mathcal{CS}, \boldsymbol{x})\) is in the core of \(G = (N, v)\). Show that for every other coalition structure \(\mathcal{CS}^\prime\) with, \begin{equation*} \sum_{C \in \mathcal{CS}} v(C) = \sum_{C^\prime \in \mathcal{CS}^\prime} v(C^\prime), \end{equation*} there is a payoff vector \(\boldsymbol{y}\) such that \((\mathcal{CS}^\prime, \boldsymbol{y})\) is in the core of \(G\).

Exercise 3: Solution

We show that \((\mathcal{CS}^\prime, \boldsymbol{x})\) is in the core of \(G\). It is immediate that we have \(x(C) \geq v(C)\) for all \(C \subseteq N\) (since \((\mathcal{CS}, \boldsymbol{x})\) is in the core of \(G\)). Thus, we need to show that \((\mathcal{CS}^\prime, \boldsymbol{x})\) is indeed an outcome - that is, we have \(x(C^\prime) = v(C^\prime)\) for all \(C^\prime \in \mathcal{CS}^\prime\).

Exercise 3: Solution

Using a similar line of reasoning to the previous question, we get, \begin{align*} \sum_{C^\prime \in \mathcal{CS}^\prime} x(C^\prime) &= \sum_{i \in N} x_i\quad&&(\text{definition of } x(C) \text{ and } \mathcal{CS}^\prime \text{ is a partition}) \\ &= \sum_{C \in \mathcal{CS}} v(C)\quad &&((\mathcal{CS}, x)\text{ is an outcome}) \\ &= \sum_{C^\prime \in \mathcal{CS}^\prime} v(C^\prime)\quad&&(\text{by assumption}). \end{align*}

Exercise 3: Solution

As stated, we have, \(x(C^\prime) \geq v(C^\prime)\) for all \(C^\prime \in \mathcal{CS}^\prime\). If there were some \(C^\prime \in \mathcal{CS}^\prime\) such that \(x(C^\prime) > v(C^\prime)\), then we would have, \begin{equation*} \sum_{C^\prime \in \mathcal{CS}^\prime} x(C^\prime) > \sum_{C^\prime \in \mathcal{CS}^\prime} v(C^\prime) \end{equation*} contradicting the previous equation.

Exercise 4

In class, we proved that a superadditive simple game has a non-empty core if and only if it has a veto player. We also claimed that the following corollary holds: a payoff vector \((x_1, \ldots, x_n)\) is in the core of a superadditive simple game \(G = (N, v)\) if and only if \(x_i = 0\) for each player \(i\) who is not a veto player. Prove this corollary.

Exercise 4: Solution

Suppose \(i \in N\) is not a veto player, and let \(\boldsymbol{x}\) be a core payoff allocation. As \(i\) is not a veto player, there must exist some coalition \(S \subseteq N \setminus \{i\}\) such that \(v(S) = 1\). Since \(\boldsymbol{x}\) is in the core, we must have \(\boldsymbol{x}(S) \geq 1\). Additionally, given that \(\boldsymbol{x}(N) = 1\), we must have \(\boldsymbol{x}(N) - \boldsymbol{x}(S) = 0\). Thus, the payoff for all players outside of \(S\) is \(0\) - since \(i \not\in S\), we have \(x_i = 0\).

Exercise 4: Solution

Conversely, suppose we have a payoff vector \(\boldsymbol{x}\) in which all non-veto players receive a payoff of \(0\). We aim to show that this allocation lies in the core, i.e., we have for all \(C \subseteq N\), \begin{equation*} \boldsymbol{x}(C) \geq v(C). \end{equation*}

Since \(\boldsymbol{x}(N)=1\), some players must receive a positive payoff. These players cannot be the non-veto players, so the sum of payoffs to the veto players under \(\boldsymbol{x}\) is 1. Now, for any winning coalition \(C \subseteq N\), all veto players must be contained in \(C\), so we have \(\boldsymbol{x}(C) = v(C) = 1\), and so \(\boldsymbol{x}\) is a stable payoff division. For any losing coalition, we must have \(V(C) = 0\), and so the core condition is satisfied automatically.

Exercise 5

Compute the players' Shapley values and describe the outcomes in the core in the following weighted voting games:

  • \(G = [13;7,7,3,3]\);
  • \(G = [10;3,3,3,3,1,1]\).

Exercise 5: Solution

First consider a player \(i\) with weight 7; they are pivotal for any coalition of size 2, or any coalition containing the other player of weight 7. There are six permutations where \(i\) is the third player, and two more where the other player of weight 7 has come first. Thus, by symmetry, the Shapley value of the both players of weight 7 is, \begin{equation*} \frac{6+2}{4!} = \frac{1}{3} \end{equation*}

Exercise 5: Solution

By efficiency, we must have for the two players \(j,k\) with weight 3, \begin{equation*} \phi_j + \phi_k + \frac{2}{3} = 1. \end{equation*} Then by symmetry, we can conclude that the two players with weight 3 must have a Shapley value of \(\frac{1}{6}\).

Exercise 5: Solution

For the second game, first consider a player \(i\) with weight 1; they are pivotal if and only if they appear in the fourth position of a permutation, after three players of weight 3. There are four ways to choose the player of weight 3 who appears after \(i\), three ways to order the three players who appear before \(i\) and two ways to order the two players that appear after \(i\). This gives 48 permutations where player \(i\) is pivotal. By symmetry, the Shapley value of both the players of weight 1 is, \begin{equation*} \frac{48}{6!} = \frac{1}{15}. \end{equation*}

Exercise 5: Solution

By efficiency, we must have for the four players \(j_1, j_2, j_3, j_4\) with weight 3, \begin{equation*} \phi_{j_1} + \phi_{j_2} + \phi_{j_3} + \phi_{j_4} + \frac{2}{15} = 1. \end{equation*} Then by symmetry, we can conclude all the players with weight 3 must have a Shapley value of \(\frac{13}{60}\).

Exercise 5: Solution

In both games, the core is empty, as there is no veto player.

Exercise 6

Consider a weighted voting game \(G = [q; w_1, \ldots, w_n]\). Prove that \(w_i \leq w_j\) implies \(\phi_i(G) \leq \phi_j(G)\).

Exercise 6: Solution

Consider a permutation \(\pi\) such that \(i\) is pivotal for \(\pi\). We aim to show that \(j\) is pivotal for the permutation obtained from swapping \(i\) and \(j\) in \(\pi\). We denote this permutation by \(\pi_{i \leftrightarrow j}\). Let \(C\) be the predecessors of \(i\) in \(\pi\). Then \(w(C) < q\), and \(w(C \cup \{i\}) \geq q\). We have two cases to consider - \(j \in C\) and \(j \not\in C\).

Exercise 6: Solution

If \(j \in C\), then the set of predecessors of \(j\) in \(\pi_{i \leftrightarrow j}\) is \((C \setminus \{j\}) \cup \{i\}\). Then we have, \begin{align*} w((C \setminus \{j\}) \cup \{i\}) &= w(C) - w_j + w_i \\ &\leq w(C) \\ & < q, \end{align*} and \begin{align*} w((C \setminus \{j\}) \cup \{i\} \cup \{j\}) &= w(C) - w_j + w_i + w_j \\ &= w(C) + w_i \\ & > q. \end{align*} Thus, \(j\) is pivotal for \(\pi_{i \leftrightarrow j}\).

Exercise 6: Solution

Now suppose that \(j \not\in C\). Then the set of predecessors of \(j\) in \(\pi_{i \leftrightarrow j}\) is \(C\), and we have, \begin{equation*} q < w(C \cup \{i\}) \leq w(C \cup \{j\}). \end{equation*} Thus, \(j\) is pivotal for \(\pi_{i \leftrightarrow j}\).

Exercise 6: Solution

Now note that for fixed \(i, j\), the mapping \(\pi \to \pi_{i \leftrightarrow j}\) is a bijection. So the number of permutations for which \(j\) is pivotal is at least as large as the number of permutations for which \(i\) is pivotal. This implies that \(\phi_i(G) \leq \phi_j(G)\).
